# PCOS Prediction Project

This repository contains code and analysis for predicting Polycystic Ovary Syndrome (PCOS) using clinical and biochemical data.

## Dataset
The dataset is available on Kaggle: [PCOS Dataset](https://www.kaggle.com/datasets/prasoonkottarathil/polycystic-ovary-syndrome-pcos?resource=download).  
Download the CSV file (`PCOS_infertility.csv`) and place it in the project root folder.

## Project Structure
- `PCOS_analysis.ipynb`: Main notebook with data cleaning, exploratory analysis, model training, evaluation, and plots.
- `requirements.txt`: Python dependencies to reproduce the analysis.
- `images/`: Saved plots such as histograms and feature importance.
- `README.md`: This file with instructions for replication.
